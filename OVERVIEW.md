# LabZoon 项目概览

本文档提供了 LabZoon 项目的高层次概述，帮助新贡献者和用户快速理解项目的架构、功能和技术栈。

## 项目简介

LabZoon 是一个基于 AI 的 Duolingo 风格学习应用，专为医学实验动物学课程设计。它提供互动式学习体验、智能题目生成和个性化学习路径，帮助用户以游戏化的方式学习实验动物学知识。

## 核心功能

### 🎯 学习系统
- **结构化课程** - 单元-课程-题目的层级结构
- **互动测验** - 支持单选、多选、判断、填空等题型
- **游戏化进度** - 经验值、连续学习天数、星级评价
- **知识点浏览** - 结构化展示学习内容

### 📚 私人题库
- **多题库管理** - 用户可创建多个私人题库
- **章节组织** - 按主题或课程组织题目
- **AI生成** - 输入学习材料自动生成题目
- **个性化练习** - 像公共课程一样练习私人题目

### 🔧 管理功能
- **用户管理** - 查看和管理注册用户
- **题库管理** - 管理公共题库和课程内容
- **数据分析** - 学习进度统计和用户行为分析
- **数据备份** - 备份和恢复重要数据

## 技术架构

### 整体架构
```
浏览器 (React) → API服务器 (Express) → Supabase数据库
```

### 前端技术栈
- **React 19.2.0** - 现代化UI框架
- **TypeScript 5.8.2** - 类型安全的JavaScript
- **Vite 6.2.0** - 快速构建工具
- **TailwindCSS 3.4.1** - 实用优先的CSS框架

### 后端技术栈
- **Express.js 4.18.2** - API服务器
- **Supabase** - 认证、数据库和实时订阅
- **JWT** - 安全认证

### AI集成
- **Google Gemini** - 主要AI内容生成服务
- **OpenAI GPT** - 备用AI服务

## 项目结构

```
labzoon/
├── components/          # React组件
│   ├── common/         # 通用组件（UI元素）
│   ├── features/       # 功能特定组件
│   └── layout/         # 布局组件
├── hooks/              # 自定义React Hooks
├── utils/              # 工具函数
├── services/           # API服务
├── server/             # 服务器端代码
├── sql/                # 数据库脚本
├── docs/               # 项目文档
├── types.ts            # TypeScript类型定义
├── App.tsx             # 主应用组件
└── main.tsx            # 应用入口点
```

## 数据库设计

### 核心表结构
- **units** - 学习单元
- **lessons** - 课程
- **challenges** - 挑战/题目
- **user_progress** - 用户学习进度
- **knowledge_items** - 知识点
- **admins** - 管理员
- **user_question_banks** - 用户私人题库
- **user_questions** - 用户私人题目

### 安全设计
- 行级安全策略（RLS）
- 用户权限分离
- 敏感信息加密
